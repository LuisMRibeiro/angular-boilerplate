Angular-boilerplate

A boilerplate to start an AngularJS project

Download this project as a .zip file Download this project as a tar.gz file

Installation

$ cd path/to/your/repo
$ git clone git@github.com:LuisMRibeiro/angular-boilerplate.git
$ npm update
$ bower update

Open the root page to see the project

Node modules

  • bower
  • shellJS

Bower components

  • angular
  • angular-http-auth
  • angular-resources
  • angular-route
  • angular-translate (Read this tutorial to learn how to use it)
  • bootstrap
  • font-awesome
  • jquery
  • modernizr
